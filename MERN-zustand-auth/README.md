# MERN-Zustand-Auth

## 📖 Description

I built an auth application using the MERN stack (MongoDB, Express, React, and Node.js).

The application uses TypeScript for both the Backend and Frontend, and uses Zustand as library for state management.

---

## 🛠️ Tech Stack

- TypeScript
- Node.js
- Express
- Zustand

---

## ⚙️ Getting Started

This project requires two terminal windows: one for the server and one for the client.

### Prerequisites

- Node.js

### Steps

1. In the root directory, start the backend server:
   ```bash
   npm run dev

2. Open a second terminal, navigate to the client folder, and start the frontend:
   ```bash
   cd client
   npm run dev
